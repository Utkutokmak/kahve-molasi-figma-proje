# Kahve Molası — Figma UI/UX Tasarım Projesi

SoftwarePersona Yazılım Stajı — **UI/UX & Web Tasarımı: Figma** proje teslimi.

Mobil platform için tasarlanmış, sıcak/soğuk kahve ve tatlı satışı yapan "Kahve Molası" adlı
kafe sipariş uygulamasının 3 ekranlık (Login → Home → Detail) arayüz tasarımı.

🔗 **Figma dosyası (canlı):** https://www.figma.com/design/XE8RQ9OrjYCIYYnuwvOZRd

## İçerik

- `UtkuTokmak_FigmaUIUX_Tasarim.pdf` — Login, Home, Detail ekranları (sıralı)
- `UtkuTokmak_FigmaUIUX_Aciklama.docx` — Proje tanımı, renk seçimi, tipografi, Figma özelliklerinin kullanım amacı
- `1-login.png`, `2-home.png`, `3-detail.png` — Ekranların yüksek çözünürlüklü görselleri
- `real-photos/` — Kartlarda kullanılan, Unsplash License ile telifsiz kahve/tatlı fotoğrafları

## Kullanılan Figma Özellikleri

- **Auto Layout**: Tüm bileşenler ve ekranlar
- **Component**: Button, Input, ProductCard
- **Variant**: Button (Default/Disabled), Input (Default/Focused)
- **Renk sistemi**: 60-30-10 kuralı, kahve kahverengisi + terracotta + adaçayı yeşili
